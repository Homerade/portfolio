PROCESSING.JS

The following are works made with processing.js, the very first language I learned when starting to code. This was my first introduction to the principles of javascript where common js methods were used to build environments including creating and maneuvering objects in the plane. Processing is a visual language often used for building playable creative pieces. These examples are a few experimentations in writing graphic and interactive code which reach to connect in response to the user outside and materials imported in.

PROJECT1

An exercise in creating a changing visual plane. The main building blocks in this project and central concepts to be grasped were the setting up of arrays, loops and looping over array items, creating new objects and pushing them into the array, as well as the creation of functions outlining movement and setting them in motion.

PROJECT 2

These sketches are plays on randomness and jolted, spasmic displays of the natural and digital together. Pixel manipulation and brightness and color mutation are used in reaction to imported video. The video, spliced together from cuts of internet finds, features different examples of lightning, providing the first level of dramatic light and darkness which the digital layer then amplifies.

PROJECT 3

Enabling user interaction through cursor watching and color and shape changes dependent on cursor position. The result is a manipulated windchime following the user's movements and introducing color on user keyboard input.
