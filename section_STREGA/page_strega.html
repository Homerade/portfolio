STREGA

Strega is a custom website and e-commerce app currently being built in Vue.js. It features a backend SQLite3 database storing products and makes use of vuex, vue router, vue-select, sequelize, node.js and express. This is the first time I've made use of Bootstrap instead of my own CSS and has seed login and registration features with sessions and passport.

STAGE1

The first step has been work on a backend admin page where products can be added to the database and then edited once entered. Forms take in the new and edited products which then fire custom functions handling the http requests, reset fields, errors and other checks. Products in the database are set to the state and passed in through an axios get request.
