Bmore Vote

Bmore Vote is a project focusing on public opinion and engagement in municipal activity. In practice it would allow city residents to know about key measures passing through the city council, understand them better and voice opinion in otherwise closed decision making.

WARNING

Baltimore does not currently have their API fully setup.

Temporarily subbing in Seattle data to populate the app.

STAGE1

Research into Baltimore City Council data channels, from Open Data Baltimore to the legislative process and beyond, revealed the legistar tracking system software used by the city, publicly posting highly detailed activity tracking information with API access.

STAGE 2

In order to open access to API information and make it available in the app, an API proxy is setup using node.js and axios. This allows for the construction of our app skeleton, funneling in the API information, custom selecting desired data, re-naming it and setting up our url parameters.

STAGE 3

Front end built in my first go with react.js. Unstyled, pure data delivery version with react router providing the main list of legistar matters shown with select info and clickable links leading to individual matter routes served up via id parameters. Voting 'for' or 'against' made possible on individual matter pages.
